# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-02-09

### Added

- Gallery (home) and Soundboard routes.
- Pre-recorded tunes (Stairway, Bounce, Sunset, Chime) with play and key hints.
- Sound (kit) selection: Bleeps, Drums, Pads with descriptions.
- Mood (scale) selection: Pentatonic, Major, Minor, Chromatic with descriptions.
- Record, play, stop, and loop for custom sequences.
- Full-screen canvas visuals tied to notes and mood.
- Touch-friendly key grid for mobile/tablet.
- Trippy neon theme (cyan, magenta, violet) with glass panels and glows.
- Scrollable options panel for Sound/Mood on small viewports.

### Tech

- React 19, TypeScript, Vite 7, React Router 7.
- Web Audio API for synthesis; Canvas 2D for visuals.
