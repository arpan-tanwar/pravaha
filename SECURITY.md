# Security

If you discover a security vulnerability in this project, please report it responsibly.

## Reporting

- **Do not** open a public GitHub issue for security-sensitive bugs.
- Email the maintainers (or open a private security advisory on GitHub if the repo supports it) with a description of the issue and steps to reproduce.

## Scope

- Security issues in this codebase (e.g. XSS, unsafe use of Web APIs, dependency vulnerabilities) are in scope.
- Issues in third-party dependencies should be reported to the respective project; we will update dependencies when fixes are available.

We will acknowledge your report and work on a fix as appropriate.
